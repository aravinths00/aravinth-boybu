A [Flarum](http://flarum.org) extension. private tags

## Installation

Install with composer:

```sh
composer require aravinth/boybu:"*"
```

## Updating

```sh
composer update aravinth/boybu:"*"
php flarum migrate
php flarum cache:clear
```
